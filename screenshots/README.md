# Folder Screenshots

Silakan letakkan berkas gambar tangkapan layar (screenshot) aplikasi Anda di folder ini:

1. Simpan screenshot tampilan desktop Anda dengan nama **`desktop.png`** di folder ini.
2. Simpan screenshot tampilan mobile Anda dengan nama **`mobile.png`** di folder ini.

Setelah berkas diletakkan di sini, berkas utama `README.md` di luar folder ini akan secara otomatis memuat dan menampilkan gambar tersebut saat dibuka di GitHub.
